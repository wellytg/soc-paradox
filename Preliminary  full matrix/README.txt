# SOC Tier 1 Research - LLM Agreement Dashboard (Standalone Export)

## Overview
This folder contains the standalone presentation version of the LLM Agreement Dashboard. 

## Files
1. **SOC_T1_Agreement_Dashboard.html**: The primary dashboard. All data (from ChatGPT, DeepSeek, Gemini, and Grok) is embedded directly into this file. 
   - No internet connection is required (except for Bootstrap CDN, which is cached by most browsers).
   - No local web server is required.
   - Simply double-click to open in any modern web browser.

2. **comparison_web.json**: The raw data source in JSON format for reference or further analysis.

## Usage
- Open `SOC_T1_Agreement_Dashboard.html` in Chrome, Firefox, or Edge.
- Use the tabs at the top to switch between Roles, Tasks, Themes, Metrics, Mappings, and Gaps.
- The color coding (Green/Yellow/Red) indicates the level of agreement between the 4 LLM agents.
